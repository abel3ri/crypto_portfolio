## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abel3ri/crypto_portfolio.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd crypto_portfolio
   ```
3. **Install dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the app:**
   ```bash
   flutter run
   ```
